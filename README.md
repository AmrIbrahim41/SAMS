# SAMS Banking Simulation — Landing Page

Official landing page for the **SAMS Bank Simulation** project at **Sadat Academy for Management Sciences**.

A bilingual (Arabic / English) single-page site featuring the academy and event logos, an animated hero, rotating inspirational quotes, feature cards, and links to all social platforms.

## Files
- `index.html` — the full page (HTML, CSS, and JS in one file)
- `college-logo.png` — Sadat Academy official logo
- `event-logo.png` — SAMS Bank Simulation 2026 logo

## Run locally
Just open `index.html` in any browser.
